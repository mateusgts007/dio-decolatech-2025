# Análise de Sentenças com IA

Este projeto foi criado para explorar o uso de IA no processamento de linguagem natural (NLP). Abaixo estão os detalhes do processo e alguns insights.

## Processo
1. Criei uma pasta chamada `inputs` para armazenar o arquivo de texto.
2. Dentro da pasta `inputs`, criei um arquivo chamado `sentencas.txt` com algumas sentenças simples.
3. Utilizei um serviço de IA (como o Azure Language Studio) para analisar as sentenças e extrair insights.

## Insights e Possibilidades
Após a análise das sentenças pela IA, aprendi que:
- A IA pode identificar **frases-chave** em um texto, como "inteligência artificial" e "processamento de linguagem natural".
- É possível realizar **análise de sentimentos** para determinar se as sentenças expressam emoções positivas, negativas ou neutras.
- A IA pode **classificar textos** em categorias, como tecnologia, educação ou meio ambiente.
- Com o uso de NLP, é possível criar **bases de conhecimento** para bots e sistemas de perguntas e respostas.


## Conclusão
A IA oferece diversas possibilidades para processar e entender textos de forma eficiente. Com ferramentas como o Azure Language Studio, é possível extrair informações valiosas e automatizar tarefas que antes exigiam esforço manual.
