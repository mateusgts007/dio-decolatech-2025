# Projeto de Reconhecimento de Texto com IA

Este projeto demonstra o uso de IA para reconhecimento de texto em imagens, utilizando ferramentas como o **Azure Computer Vision** ou **Tesseract OCR**.

## Estrutura do Projeto
- **inputs/**: Contém as imagens utilizadas para o reconhecimento de texto.
- **outputs/**: Armazena os resultados do reconhecimento, como textos extraídos e imagens anotadas.

## Processo
1. **Coleta de Imagens**: Utilizei imagens de recibos, notas fiscais e documentos para testar o reconhecimento de texto.
2. **Reconhecimento de Texto**: Usei o serviço de **Azure Computer Vision** para extrair o texto das imagens.
3. **Salvamento dos Resultados**: Os textos extraídos foram salvos na pasta `outputs`, junto com imagens anotadas (se aplicável).

## Insights e Possibilidades
- A IA é capaz de extrair texto de imagens com alta precisão, mesmo em documentos complexos.
- É possível automatizar processos como digitalização de recibos, notas fiscais e documentos manuscritos.
- A integração com outras ferramentas, como bancos de dados ou sistemas de gestão, pode ampliar as aplicações.
